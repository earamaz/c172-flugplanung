Short-Field-Technik. Korrekturen lt. POH: Gegenwind −10 %/9 kt, Rückenwind +10 %/2 kt, Gras trocken +15 % (Start) bzw. +45 % (Landung) auf die Rollstrecke. Gras nass/weich +30 % / +60 % (Faustregel, nicht POH). Kalt/leicht wird konservativ auf Tabellenränder begrenzt. Neigung: Faustregel +10 % Rollstrecke je 1 % ungünstiger Steigung (Start) / Gefälle (Landung) — nicht im POH, günstige Neigung ohne Bonus. Bahnlängen aus AIP eingeben für die „reicht / zu kurz"-Prüfung.
